# 黑盒测试文档

## 1. 测试计划

### 1.1 测试目标

- **描述**: 验证系统功能的正确性和稳定性，确保在各种输入情况下系统能够按预期工作。
- **预期成果**: 发现并修复系统中的缺陷，确保系统在发布前达到质量要求。

### 1.2 测试范围

- 包含的功能

  - 数据管理功能
  - 成绩管理功能
  - 课程管理功能
  - 就业能力评估功能
  - 就业去向预测功能
  - 实时评估和预测功能

### 1.3 测试环境

- 硬件

  - PC: Windows 10, 8GB RAM, i5处理器
- 软件

  - 浏览器: Chrome, Firefox, Safari, Edge
  - 数据库: MySQL 8.0
  - **网络**: 本地网络连接
